Revisando Exercícios

Imagem Dinâmica

Este exercício é para relembrar o quanto as imagens podem ser redimensiondas de acordo com o tamanho das telas. A medida que o tamanho do navegador vai mudando, as imagens vão mudando também.

Reproduzindo Áudio

Este exercício de áudio é para mostrar o controle de áudio e ele deve ser tocado automaticamente ao abrir ou atualizar a página.

Vídeos hospedados localmente

Este exercício é para mostrar que os vídeos hospedados localmente, também são reproduzidos se colocados em loop.

Vídeos do YouTube

O vídeo do YouTube é mais fácil de ser reproduzido se o link foi inserido de forma correta.
